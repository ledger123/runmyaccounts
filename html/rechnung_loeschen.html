Rechnung löschen - Run my Accounts Hilfe

Eine bereits verbuchte Rechnung löschen

  1. Suchen Sie die bereits verbuchte Rechnung mittels Debitoren - Berichte - Buchungen und klicken Sie die beiden Felder "Offen" und "Geschlossen" an. Grenzen Sie falls gewünscht die Rechnungen weiter ein.
  2. Klicken Sie auf den Knopf "Weiter".
  3. Klicken Sie auf die Rechnungsnummer der gesuchten Rechnung.
  4. Drücken Sie unten auf den Knopf "Löschen" und bestätigen Sie mit einem Klick auf den Knopf "Weiter", dass die entsprechende Rechnung tatsächlich gelöscht werden soll.
