Akontorechnung erfassen - Run my Accounts Hilfe

Akonto Artikel erfassen

  1. Erfassen Sie einen Artikel "Akonto" nach der Anleitung zu Artikel erfassen.
  2. Als Ertragskonto für den Artikel verwenden Sie das Konto "Sonstiger Ertrag" oder allenfalls ein Konto "Akonto Zahlungen".

Akontorechnung erfassen

  1. Erfassen Sie eine Rechnung analog zur Anleitung Rechnung efassen und verwenden Sie den oben erstellten Akonto Artikel. Passen Sie den Preis (=Akontozahlung) nach Bedarf an. 

Schlussabrechnung erstellen

  1. Erfassen Sie die Schlussabrechnung analog zur Anleitung Rechnung efassen. Verwenden Sie dazu die Artikel, die Sie dem Kunden verkauft haben. 
  2. Verwenden Sie erneut den Akontoartikel, z.B. als letzten Posten auf der Rechnung. Als Preis nehmen Sie den negativen Betrag der Akontozahlung. Somit wird die Akontozahlung dem Rechnungsbetrag in Abzug gebracht.
