Artikel erfassen - Run my Accounts Hilfe

Dienstleistungen erfassen

  1. Sie erfassen Ihre Dienstleistungen, indem Sie Waren & Dienstleistungen - Dienstleistung erfassen aufrufen.
  2. Falls Sie einen Artikelstamm mit einer bestimmten Nummerierungsmethode führen, geben Sie die gewünschte Artikelnummer ein. Andernfalls wird vom System eine Artikelnummer generiert. 
  3. Unter Beschreibung geben Sie den Namen des Artikels an.
  4. Unter Erneuert am geben Sie das Datum ein, ab dem der Artikel zur Verfügung stehen soll. 
  5. Das Feld Verkaufspreis ist kein obligatorisches Feld. Wenn der Verkaufspreis stets ändert, empfehlen wir Ihnen, keinen Verkaufspreis zu erfassen.
  6. Unter Ertrag wählen Sie das Erfolgskonto aus, auf das der Artikel gebucht werden soll. Das Aufwand-Konto ist nur relevant, falls Sie die Dienstleistung von einem Drittanbieter einkaufen.
  7. Wählen Sie den korrekten MWST Satz der Dienstleistung aus.
  8. Unter Bemerkungen knnen Sie interne Kommentare zum Artikel erfassen.
  9. Unter Kunde können Sie einem bestimmten Kunden spezielle Konditionen zuordnen. Wenn Sie eine Rechnung an den Kunden schreiben, wird der spezifische für den Kunden geltende Verkaufspreis aus der Datenbank gelesen. 
  10. Speichern Sie den erfassten Artikeln mit einem Klick auf "Speichern". Der Artikel steht Ihnen fortan zur Verfügung.
